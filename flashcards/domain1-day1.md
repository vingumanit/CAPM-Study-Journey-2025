# CAPM Domain I Flashcards (20 Terms) - Day 1

**AI Prompt**: "20 Anki flashcards: CAPM Domain I terms (front: term, back: definition+example)"

## Front → Back Format (Copy to Anki)

**1. Project Lifecycle**  
*Phases from start to closeout.* Ex: Agile (sprints), Predictive (phases) [web:141]

**2. Predictive Lifecycle**  
*Traditional waterfall: sequential phases.* Ex: Construction (design→build)

**3. Agile Lifecycle**  
*Iterative/incremental delivery.* Ex: Software dev (2-week sprints)

**4. Hybrid Lifecycle**  
*Combines predictive + agile.* Ex: Marketing (plan + iterate)

**5. Tailoring**  
*Adapting processes to context.* Ex: Agile for software, waterfall for hardware

**6. PM Principles**  
*12 universal PM behaviors (PMBOK 7).* Ex: Stewardship, value-focused

**7. Project Manager**  
*Responsible for project success.* Leads team, manages stakeholders

**8. Project Sponsor**  
*Provides funding/authority.* Approves charter, resolves issues

**9. Process Groups**  
*5 groups: Initiating→Planning→Executing→M/C→Closing*

**10. Initiating**  
*Authorize project.* Create charter, identify stakeholders

**11. Planning**  
*Define scope/schedule/budget.* Develop baseline plans

**12. Executing**  
*Manage team/deliver work.* Direct resources

**13. Monitoring/Controlling**  
*Track performance vs plan.* Manage changes

**14. Closing**  
*Finalize deliverables.* Obtain acceptance

**15. Stewardship**  
*Principle 1: Diligent/respectful caretaker.* Ex: Resource protection

**16. Value-Focused**  
*Deliver maximum value.* Prioritize benefits

**17. Stakeholders**  
*Anyone impacted by project.* Engage throughout

**18. System Interactions**  
*Principle 5: Understand connections.* Ex: Dependencies between teams

**19. Leadership Behaviors**  
*Principle 6: Inspire/motivate.* Servant leadership

**20. Collaborative Environment**  
*Principle 2: Teamwork culture.* Psychological safety
